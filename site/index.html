MYTHODIC

MYTHODIC is a public-facing starter package for an overlap-intelligence plugin: a processor built to decide whether stacked sounds should separate, fuse, or crown, so layered productions feel wider, cleaner, and more intentional.

Read the README View roadmap DSP design

Core idea

Treat overlap itself as the musical event instead of fighting it with blunt EQ, width, or saturation.

Current stack

analysis → masking → decisions → oversampled tone → space → room → safety

Status

Starter repo. Real architecture, public docs, repo hygiene, and a compile-oriented JUCE baseline.
