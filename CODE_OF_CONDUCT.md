# Code of Conduct

## Our Pledge
We want this project to be open, technically sharp, respectful, and useful.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- discussing ideas and implementation details constructively
- grounding claims in measurable behavior
- reporting bugs with reproduction details
- respecting contributors across skill levels

Examples of unacceptable behavior include:
- harassment, abuse, or discrimination
- personal attacks or bad-faith reviews
- deceptive technical claims
- spam or low-effort promotion in issues and pull requests

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, code, wiki edits,
issues, and other contributions that are not aligned with this Code of Conduct.
